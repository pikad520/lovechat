# Feature Specification: 本地语音朗读（Kokoro TTS）

**Feature Branch**: `005-voice-tts`

**Created**: 2026-06-12

**Status**: Draft

**Input**: User description: "集成 Kokoro TTS 并随安装包发布；设置里勾选语音后自动检测依赖，缺失则一键下载自动安装，完成即用。已选定路线：sherpa-onnx 原生 ONNX 集成。"

## User Scenarios & Testing

### User Story 1 - 开启语音与一键安装 (Priority: P1)

用户在设置「语音」页打开语音开关；应用检测到本地模型缺失，显示模型说明（体积、来源）
与「下载语音模型」按钮；点击后展示下载进度，完成后自动解压就绪，无需重启即可使用。

**Acceptance Scenarios**:

1. **Given** 全新安装、首次打开语音开关，**When** 检测到模型缺失，**Then** 显示一键下载按钮与模型体积说明，语音功能暂不可用但不报错。
2. **Given** 点击下载，**When** 下载进行中，**Then** 显示进度（百分比/已下载量），可取消；App 其他功能不受影响。
3. **Given** 下载完成，**When** 自动解压校验通过，**Then** 状态变为「已就绪」，播放按钮立即可用。
4. **Given** 下载中断/失败，**When** 用户重试，**Then** 重新下载，不留损坏的半成品文件。
5. **Given** 模型已就绪，**When** 重启应用，**Then** 不再提示下载，开关状态保持。

---

### User Story 2 - 消息朗读 (Priority: P1)

语音就绪后，每条角色消息旁出现 🔊 按钮，点击用所选音色朗读该消息正文
（跳过括号内的心理活动）；再次点击停止。

**Acceptance Scenarios**:

1. **Given** 语音就绪，**When** 点击角色消息的 🔊，**Then** 本地合成并播放，无网络请求。
2. **Given** 正在播放，**When** 再次点击或点击其他消息的 🔊，**Then** 停止当前播放（后者切换到新消息）。
3. **Given** 消息含（心理活动），**When** 朗读，**Then** 括号内容被跳过。
4. **Given** 模型未就绪，**When** 查看消息，**Then** 🔊 置灰，悬停提示去设置下载。

---

### User Story 3 - 音色与自动朗读 (Priority: P2)

设置中可选全局默认中文音色（女声/男声若干，可试听）；角色编辑中可为角色指定音色
（默认跟随全局）并开启「自动朗读」——开启后该角色每条回复完成时自动朗读。

**Acceptance Scenarios**:

1. **Given** 语音设置页，**When** 选择音色并点试听，**Then** 播放一段示例语音。
2. **Given** 角色开启自动朗读，**When** 角色回复流式完成，**Then** 自动开始朗读该回复。
3. **Given** 自动朗读进行中，**When** 用户发送新消息，**Then** 朗读停止，让位于新一轮对话。

---

### User Story 4 - 外接语音服务（GPT-SoVITS 等） (Priority: P3)

进阶用户自行部署 GPT-SoVITS（或任何 OpenAI 兼容 TTS 服务），在设置中添加外接语音
服务（名称、地址、协议类型；GPT-SoVITS 还可配置参考音频与提示文本实现克隆音色）；
角色可绑定该服务的音色，朗读时走外接服务，获得更高音质与专属声线。

**Acceptance Scenarios**:

1. **Given** 本地跑着 GPT-SoVITS api_v2，**When** 添加外接服务并测试，**Then** 反馈连通结果。
2. **Given** 角色绑定外接音色，**When** 朗读该角色消息，**Then** 请求外接服务合成播放。
3. **Given** 外接服务不可达，**When** 朗读，**Then** 自动回退内置引擎（若就绪），否则轻量提示；文字对话不受影响。

---

### Edge Cases

- 推理失败（模型损坏等）→ 轻量提示 + 建议重新下载；文字对话完全不受影响（宪法 V）。
- 磁盘空间不足 → 下载前预检并提示。
- 超长消息 → 按句切分顺序合成播放，首句出声即开始。
- 删除模型（设置中提供入口）→ 释放磁盘，状态回到未就绪。

## Requirements

- **FR-401**: 语音合成 MUST 完全本地推理（sherpa-onnx + Kokoro 多语言 ONNX 模型），运行期无网络调用、无独立服务进程。
- **FR-402**: 模型 MUST NOT 随安装包分发；设置中开关 + 自动检测 + 一键下载（带进度、可取消、失败可重试、完整性校验），下载至 Application Support。
- **FR-403**: 推理库以 vendored 预编译产物入库（宪法 II 豁免条款），封装隔离于独立服务层；模型缺失/失败时优雅降级，不影响文字功能。
- **FR-404**: 角色消息提供播放/停止控制；朗读跳过心理活动括号内容；同一时刻仅一路播放。
- **FR-405**: 提供中文预置音色选择（含试听）；角色级音色覆盖 + 「自动朗读」开关。
- **FR-406**: 设置中可查看模型占用并删除模型。
- **FR-407**: 支持外接语音服务，协议二选一：OpenAI 兼容 `/v1/audio/speech`、GPT-SoVITS `api_v2 /tts`（含参考音频/提示文本配置）；提供连通性测试。
- **FR-408**: 角色级音色可指向内置音色或外接服务音色；外接失败回退内置（若就绪）。

## Success Criteria

- **SC-401**: 从勾选语音到可朗读，用户操作不超过 2 次点击（开开关 + 点下载）。
- **SC-402**: 短句（≤50 字）从点击到出声 ≤ 2 秒（Apple Silicon）。
- **SC-403**: 语音全流程任何失败都不影响文字对话（零回归）。
- **SC-404**: 安装包体积增量 ≤ 30MB（仅推理库，模型不打包）。

## Assumptions

- 模型托管：sherpa-onnx 官方 GitHub Releases 的 kokoro 多语言模型包（含中文音色）；
  如下载源不稳定可镜像至本项目 Release。
- 仅支持 Apple Silicon（与现有发布包一致）。
